What is the web?

The web connects my computer with many others through the exchange of data.
The data are organised in html files that bounce back and forth from servers through a protocol called HTTP, or Hypertext Transfer Protocol .

This video is a good starting point.

What is HTML?

HTML stands for Hypertext Markup Language. HTML is the language used to build pages that are visualised when a computer connects to a server.
A server is a central computer capable of storing all the information visualised by the computers connected to it through the HTTP protocol.

HTML Attributes

HTML documents include attributes, that make the computer do stuff such as including links in the page. An example is the HREF attribute, used for links. The string of code looks like this:

open brackets, A, href= "link", close brackets,some string of content that will be attached to the link, and then closing tag /A in brackets

It's never the machine, it's you

This is a popular say among programmers here in London. It means that whatever mistake your page shows, it's never the computer's fault.
Mistakes are generated by problems in your code that you will have to debug.
In fact, a computer interprets your instructions in a literal way, each sign corresponds to one operation that never changes. There is no space for nuances.

Void tags

Void tags are called so because they don't have any content, hence they don't need to be opened and closed. So for example, the tag:

'BR'
Can be put in brackets and there's no need to open an close it, it's just a break in the text.


Inline and block elements

Inline elements in an HTML page are elements that add or do something within a chunk of code, without separating it from the rest.
Block elements, on the other hand, create an invisible block that sets apart different parts of your code.
Noticeably, using an inline or a block element can sometimes produce similar results, but the ratio behind the operation is very different.
For example, the tag BR that we've seen earlier. Its purpose is just to break a line.
If we use the tag P, which stands for 'paragraph', the result may seem the same: a break in the paragraph. But the function is very different. The tag P creates a block around the paragraph, that will function - and display, depending on the CSS applied - very differently from the tag BR.
